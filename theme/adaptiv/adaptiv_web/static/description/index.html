Make it yours

Multi company branding

You know which colors you love the most. That's why the Adaptiv backend theme gives you full control over it's color palette.

  • Create your own color palette from the settings panel.
  • Support for light and dark versions of your logo.
  • Support for both SVG and PNG file formats.
  • New optional debrand features (including custom favicon).

Get the most out of your screen area

Meet the shell overlay.

The shell overlay replaces every large dialog found in Odoo and displays content on the side for an optimized experience. At small screen sizes the shell panes will completely cover the screen.
